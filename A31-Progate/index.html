Ayo Kita Mulai

  • Apa itu DTS20 ?

    DTS20 adalah program kerjasama Kominfo dengan beberapa pihak untuk melaksanakan sejumlah program yang dicanangkan oleh Kominfo

    Program ini sudah berlangsung selama beberapa tahun dan untuk tahun ini ditandai dengan kode DTS20 (Program DTS tahun 2020)

    Untuk detail program yang tersedia bisa klik DTS Kominfo

  • Untuk Program HTML, CSS dan JavaScript, Kominfo menggandeng Progate sebagai partner

    (Klik PROGATE untuk detail)

    Aktifitas dalam program ini meliputi:

    • Self Pace/Belajar Mandiri dengan dibantu Tutor
    • Live Session/Webinar
    • Virtual Lab
    • Grup Kelas
    • Ujian/Kuis
    • Certificate of Completion
